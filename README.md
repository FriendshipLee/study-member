# SpringBoot를 이용한 간단한 회원기능 구현
  > 유튜브 '코딩레시피' 영상 참고

-----

## 기능
  + 회원 가입
  + 로그인 / 로그아웃
  + 회원 수정 / 탈퇴

-----

## 개발환경
  + Open JDK 17
  + IntelliJ IDEA Ultimate
  + MySQL Server 8.0
  + Spring Boot 3.5.3

-----

## 라이브러리 / 프레임워크
  + Spring Data JPA
  + Thymeleaf
  + lombok
  + gradle
  + jquery
